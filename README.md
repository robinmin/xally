# X-Ally

#### Usage
```bash
$ ./build/bin/xally
```

#### Installation

```bash
$ go install github.com/robinmin/xally@latest
$ cd xally
$ make build
```
> mkdir -p build/bin
> GO111MODULE=on go build -o build/bin/xally ./cmd/client/main.go
> chmod u+x build/bin/xally
